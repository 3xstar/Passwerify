# Passwerify
First PyQt project


PROJECT TOPIC: an application for checking passwords for their reliability in various aspects, such as:

Password length
Presence of capital letters
Presence of lowercase letters
Presence of numbers
Presence of special characters
Absence of common or weak combinations
Absence of user's personal information
Uniqueness (absence of reuse)
Dictionary check for common passwords
Entropy-based complexity


THE PURPOSE OF THE PROJECT:
emulate a deep password check so that the user can test their passwords for strength and then see which aspects of the check their password passes and which it fails
